# Cielo Obs RD

Aplicación móvil para registrar observaciones del cielo (fenómenos atmosféricos, astronómicos, avistamientos, etc.). Construida con Expo + React Native + TypeScript y almacenamiento local en SQLite.

La aplicacion permite agregar una observacion y permite insertar extra informacion como imagen, coordenadas y otras informaciones.

Ademas permite listar todas las observaciones que se agregaron.

## Como instalar

Para instalar la aplicacion se debe instalar el APK en este enlace: [Descargar APK](https://drive.google.com/drive/u/1/my-drive)
